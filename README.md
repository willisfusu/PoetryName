# 你的名字-用诗经, 楚辞, 唐诗,宋词起名字

## demo

- [点击查看demo](http://xiaosang.net/gushi_namer/), 支持手机查看

## AI 起名说明

- 当前版本支持通过 AI 基于诗文内容直接生成名字候选列表。
- 可在页面右上角「设置」中配置 API Key。
- 不填写自定义诗文时，会基于当前所选诗文库生成；填写自定义诗文时优先使用自定义内容。
- 每个候选名字会展示对应诗句摘录与命名理由，并支持直接选择。

## 源码运行方法

- 方法1: 直接点击`dist`目录中压缩好的html文件
- 方法2: 源代码运行

```
git clone https://github.com/holynova/gushi_namer.git
npm install

# 本地调试
npm start

# 编译
npm run build

```
